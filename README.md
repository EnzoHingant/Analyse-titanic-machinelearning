# Titanic Machine Learning 

## Description 
Ce projet consiste à construire un modèle de machine learning capable de prédire la survie des passagers du titanic. 

## Objectif 
Prédire la variable "Survived" à partir de caractéristiques comme : 
- L'âge
- le sexe
- la classe sociale

## Technologies 
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Méthodologies 
- Nettoyage de données
- Encodage des variables
- séparation test
- Entrainement d'un modèle de regression logistique
- Evaluation de performances

## Résultats 
Le modèle obtient une précision d'environ 74%.

 Les variables les plus influentes sont : 
 - Le sexe
 - La classe sociale

## Conclusion 
Ce projet illustre la mise en place d'un pipeline de machine learning simple et l'impact des variables sur la prédiction.
